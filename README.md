CKP-CMP_Simulator 

Simulador de señales CKP/CMP portátil con Arduino y LCD Keypad Shield para banco de pruebas de ECUs. Replica digitalmente ondas cuadradas (0-5V) de 6 marcas (VW, Ford, Peugeot). Permite seleccionar la rueda fónica con botones y regular RPM (600-6000) con potenciómetro para diagnosticar fallas en computadoras automotrices de forma segura.
